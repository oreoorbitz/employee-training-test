Javascript is a programing language specifically made to enable more interactivity on the web above the default function of html and css. 
As such javascript is a very event focused language, though that is not an official definition.
The most common type of application of javascript will be, first, identifying HTML elements and saving them in memory inside of a the javascript runtime, second, 
subscribing to events that involve those HTML elements, third, executing a or multiple functions.

In this module, store a DOM element as a javascript variable, then create an eventListener to react to the clicks on that DOM element. 
Lastly, when that DOM element is clicked, execute a function
